    这道题目实际上不难，然而毕竟自己第一次打kick start，也好久不碰算法题了。在这里记录一下

    题目描述

    具体原文背景就不再翻译了，题目大意是：给定一组严格递增的数据，可以在其中插入数据，同时要保证数列仍然是严格递增的，最多插入$K$个新数据。问最终的最大间隔最小值是多少。

    输入数据格式：T表示测试数据组数，紧接着每一组数据先输入$N$，$K$表示初始有$N$个数据，最多能插入$K$个。接下来输出N个初始数据$X_i$。

    输出：最大间隔的最小值。

    题解

    第一个点仅仅要求$K=1$，这是很容易的，因为此时唯一一种选择就是将最大区间对半分。然而如果顺着这个思路往下走，就会像我一样误入歧途了。因为当$K>1$的时候，显然一直对半分是不正确的。例如初始数据是$1, 13$且$K=2$，那么对半分两次得到的结果是6，不如三等分。因此要决定每个区间分多少份是一件不太好做的事情，而我一直在想这个东西能不能dp……就失败了。

    正确的做法是二分（似乎当你不知道怎么办的时候二分就对了.jpg）。假设初始所有间隔为$d_1, d_2, …d_{N-1}$，那么如果想要得到一个结果为$ans$的分割方式，那么对每个$d_i$至少要切$\lceil \frac{d_i}{ans} \rceil + 1$次，将其加起来如果不大于$K$，那么说明$ans$是可行的，否则是不可行的。因此对$ans$进行二分即可找出答案。
